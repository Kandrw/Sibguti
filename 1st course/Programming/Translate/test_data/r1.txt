Чтобы open текстовый file on record, необходимо применить режим w (перезапись) either a (дозапись). 
Затем for записи применяется метод write(str), 
        in which передается записываемая строка. 
Стоит отметить, what записывается именно строка, therefore, 
        if нужно записать числа, данные других типов, 
    то их предварительно нужно конвертировать in строку.
